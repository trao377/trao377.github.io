I. Le chat

Pour le chat, il faut un UI classique. Pas besoin de s'embêter la vie pour faire un truc méga-complexe.

Personnellement, je change globalement les préfixes de mes messages à chaque projets. Voici des exemples :

Hypixel » Mon message...
[Hypixel] Mon message…
Hypixel: Mon message…
Hypixel - Mon message…

Pour les couleurs ; je conseille de faire quelques choses de simpliste et clair par exemple qui est lié au thème de votre serveur. Tout dépend de l'utilisation.

II. Les BossBar / ActionBar

Pour ces derniers, faut être toujours aussi simple. Si vous êtes en 1.9 + vous pouvez mettre de la couleur sur la barre. Sinon si vous êtes en 1.8 cela sera en violet.

Personnellement, je conseille de faire un /annonce comme à la Rinaorc car cela était une très bonne idée. Si vous êtes en 1.9 et + ; vous pouvez choisir la couleur de la bossbar si l'annonce est accepté.

    ItemActions.set(stack, MY_ACTION);
    // give item here
}

    String action = ItemActions.get(stack);

}

@EventSubscribe
void onItemAction(ItemActionEvent event) {
    if (MY_ACTION.equals(event.getAction())) {
        event.markHandled();

ItemConstraints

Malheureusement, je n'ai pas d'exemples propre à montrer; mais voici :

ItemStack stack = [...];
ItemConstraints.lock(stack);
// give

L'item ne pourra pas être déplacé du tout

ItemStack stack = [...];
ItemConstraints.link(stack);
// give

L'item est bloqué dans l'inventaire dans lequel il est placé (celui du joueur, un coffre, etc)
