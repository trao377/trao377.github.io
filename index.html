IV. Les couleurs

Pour les couleurs, tout dépend de votre thème de serveur, ou que la couleur souhaitez. Voici une palette de couleur qui vas bien ensemble:

  • Orange (&6), Jaune (&e), Bleu ciel (&b), Blanc (&f)
  • Gris (&7), Blanc (&f), Jaune (&e), Orange (&6)
  • Rose (&d), Violet (&5), Blanc (&f), Gris (&7)
  • Bleu (&9), Blanc (&f)

    ItemActions.set(stack, MY_ACTION);
    // give item here
}

void get(ItemStack stack) {
    String action = ItemActions.get(stack);
    // Do something
}

@EventSubscribe
void onItemAction(ItemActionEvent event) {
    if (MY_ACTION.equals(event.getAction())) {
        event.markHandled();
        // Do something
    }
}
